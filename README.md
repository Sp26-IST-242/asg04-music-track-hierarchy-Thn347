Overview
- This project implements an object‑oriented music track system using inheritance, polymorphism, abstract base classes, and comparison operators.
- The class hierarchy includes:
    Artist
    Album
    MusicTrack (abstract)
        Song
        Podcast
    Playlist
- The program supports formatted play times, sorting tracks by release year, and polymorphic behavior across mixed track types.

Author name: Thuan Nguyen

Project spec: 
- Uses abc for abstract methods
- Implements comparison dunder methods for sorting
- Demonstrates polymorphism through shared interfaces
- Includes defensive copying and encapsulation
- Provides a main.py script that builds and sorts a playlist
